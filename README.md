## Features

- Views windows tasks in an organized fashion from used open apps

- System Info tab - reads system information and views them in an organized readable card format.

- Network Testing tab - Allows users to view Network ups and downs. Users can also utilize Speed test from the same tab.

## Security Features

- System tasks are locked for security and overall functionality of windows system

- emptystandbylist.exe is trusted outsource that operates only when users allow RAM cleaning escalation (ADMINS ONLY)

## Cleaner Functionality

The RAM cleaner utilizes emptystandbylist.exe to perform RAM cleansing of process without impacting any vital system components.
